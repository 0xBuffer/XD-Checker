# XD Checker

**XD Checker Recode** — это мощный и интуитивно понятный инструмент для глубокого анализа системы на наличие следов использования стороннего ПО (читов) в Minecraft. Программа создана для проведения быстрых и эффективных проверок (SS — Screen Share).

---
> [!WARNING]
> ## ⚠️ Важная информация по запуску
>
> При запуске программы вы можете увидеть предупреждение Windows (**SmartScreen**) о том, что "Система Windows защитила ваш компьютер".
>
> **Почему это происходит?**
>
> 1. **Отсутствие цифровой подписи:** Программа является некоммерческим софтом и не имеет дорогостоящего сертификата разработчика.
> 2. **Специфика работы:** Для поиска скрытых читов программа выполняет глубокое сканирование дисков и чтение системных журналов (JournalTrace). Антивирусы часто помечают такое поведение как подозрительное (ложное срабатывание на "стиллер" или "троян"), так как софт обращается к низкоуровневым данным системы.
> 3. **Привилегии:** Программа автоматически запрашивает права администратора, необходимые для доступа к файловой таблице и индексам.
> 
> **Почему нам можно доверять?**
>
> * **Открытый исходный код:** Проект имеет полностью [открытый исходный код](https://github.com/0xBuffer/XD-Checker/tree/main/code). Вы или любой технический специалист можете проверить каждую строчку кода на отсутствие вредоносных функций.
> * **Прозрачность:** Программа делает только то, что заявлено — помогает найти следы читов.
>

> [!NOTE]
> ## Обзор интерфейса
>
> ### 1. Главная панель сканирования
>
>Инструмент для запуска поиска по дискам с возможностью выбора типа сканирования.
>
>### 2. Меню навигации
>
>Удобное боковое меню для быстрого переключения между модулями.
>
>### 3. Библиотека инструментов
>
>Встроенный набор утилит (AnyDesk, Everything, PrivaZer) для глубокого анализа.
>
>### 4. Системная информация
>
>Отображение подробных характеристик ПК и статуса стороннего ПО (например, OBS).
>
>### 5. Детальные параметры сборки
>
>Данные об ОС и дате установки для выявления манипуляций с системой.

---
> [!IMPORTANT]
>## Основные возможности
>
>* **Сканирование диска:** Поиск файлов по специфическим паттернам и индексам.
>* **Анализ журналов:** Отслеживание действий пользователя через JournalTrace.
>* **Восстановление истории:** Просмотр удаленных файлов с помощью PreviousFilesRecovery.
>* **Системный отчет:** Полная сводка о железе и версии Windows.
